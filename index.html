How to Set Up Your Google Sheet

  1. Create a new Google Sheet
  2. Add columns with headers: "ID", "Team Name", "Score"
  3. Add your team data
  4. Go to File > Share > Publish to web
  5. Select "Entire Document" and "CSV" format
  6. Click Publish and copy the URL
  7. Replace the spreadsheet ID in the JavaScript code

Important: For this example to work, your Google Sheet must be published to the web as CSV.
